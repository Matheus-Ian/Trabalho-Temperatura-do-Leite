# Trabalho LacTec — Relatório Completo

## 1. Introdução
Este trabalho aborda um estudo sobre monitoramento de temperatura aplicado a processos industriais, inspirado em projetos reais utilizados na cadeia do frio.

## 2. Problema
Temperaturas inadequadas podem causar perdas de qualidade e riscos sanitários. O projeto visa exemplificar um sistema de monitoramento para mitigação desses riscos.

## 3. Objetivo
Desenvolver uma estrutura conceitual contendo relatório, código e documentação organizada.

## 4. Metodologia
- Análise teórica
- Estruturação do sistema
- Criação de código demonstrativo
- Documentação do processo

## 5. Protótipo Conceitual
O protótipo utiliza leitura simulada de temperatura e impressão dos dados.

## 6. Conclusão
O trabalho demonstra como documentar, estruturar e organizar um projeto técnico de forma clara e objetiva.
